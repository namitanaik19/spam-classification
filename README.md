# Spam SMS Classifier

A Machine Learning-based Spam SMS Classifier built using Python, Scikit-Learn, NLTK, and Streamlit.

## Features
- Spam/Ham Classification
- Text Preprocessing using NLTK
- TF-IDF Vectorization
- Naive Bayes Classification
- Interactive Streamlit Web App

## Tech Stack
- Python
- Pandas
- NumPy
- Scikit-Learn
- NLTK
- Streamlit
